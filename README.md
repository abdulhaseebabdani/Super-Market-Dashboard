# Super-Market-Dashboard
Interactive visualization built to show how 3 different superstores performed in 1st quarter, based on their sales and profitability analysis.

The Overview page provides a snapshot of the business, including store locations, product hierarchy, total orders, customer segmentation, and a summarized sales & profit trend.

The Sales Analysis page delivers detailed insights through key performance indicators (KPIs) such as total sales, total orders, quantity sold, average order value, and daily average sales. It includes visualizations like monthly sales trends, product line contribution, gender-based sales comparison, tax distribution, and order patterns. A dynamic filter panel allows users to analyze data by date, time, city, store, payment method, and customer type.

The Profitability Analysis page focuses on financial performance with KPIs such as net profit, gross margin percentage, cost of goods sold, profit contribution, and average profit per order. It includes visuals like profit distribution by product line, cost vs. sales vs. profit comparison, customer-type profitability, store-level performance, and region-wise sales hierarchy.

A reset button also implemented on each page at right top for resetting all filters to default view.

The data has sourced from **Kaggle.com** with some alterations for dashboard presentation.

⚙️ Key Functions of the Dashboard: Provides a complete business overview in one place. Tracks sales performance across multiple dimensions (time, product, store).Enables customer segmentation analysis (gender, member vs normal). Allows interactive filtering for deep data exploration. Highlights top-performing and underperforming areas. Shows regional and product-level contribution. Supports decision-making with real-time insights & also includes reset functionality for quick navigation.

📊 How This Dashboard Helps Understand Data: Simplifies complex data into easy-to-read visuals. Helps identify sales trends and seasonality patterns. Reveals which product lines generate the most revenue. Shows profit vs cost relationship, improving financial understanding. Identifies high-value customers and buying behavior. Highlights which stores or regions perform better.Improves data-driven decision making for business growth and saves time by replacing manual analysis with automation.
